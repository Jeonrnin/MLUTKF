# MLUTKF Lorenz-96 experiments

This repository contains the source code, plotting scripts, and processed outputs used for the Lorenz-96 data assimilation experiments in the manuscript:

**Modulated Local Unscented Transform Kalman Filter for Nonlinear Systems**

The code includes implementations of the LETKF, LUTKF, GETKF with modulation ensemble, and MLUTKF. No real observational datasets were used; synthetic observations were generated within the Lorenz-96 experiments.

The plotting scripts were used to generate Figs. 1--12 and the result tables in the manuscript.
